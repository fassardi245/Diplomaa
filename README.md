#  SmartCloth Logistics

SmartCloth Logistics es una plataforma web integral desarrollada como proyecto final académico para la obtención del título intermedio de **Analista en Sistemas Informáticos**.

El proyecto fue realizado por un equipo de tres estudiantes durante aproximadamente un año de trabajo, abarcando tanto las etapas de análisis y diseño de software como el desarrollo e implementación completa de la solución.

La plataforma fue diseñada para gestionar de manera integral el ciclo de ventas y distribución de indumentaria, combinando funcionalidades de comercio electrónico con herramientas de logística, administración, auditoría y análisis de datos orientadas a la toma de decisiones empresariales.

---

##  Objetivo del Proyecto

Desarrollar una solución capaz de gestionar el proceso completo de comercialización y distribución de productos, desde la compra realizada por el cliente hasta la planificación logística y entrega del pedido, proporcionando además herramientas de control, monitoreo y análisis para administradores.

---

##  Análisis y Diseño del Sistema

Durante la primera etapa del proyecto se realizó un proceso completo de Ingeniería de Software, incluyendo:

* Relevamiento y análisis de requerimientos funcionales y no funcionales.
* Elaboración de casos de uso.
* Construcción del modelo de dominio.
* Diseño de diagramas de clases.
* Modelado de la base de datos mediante diagramas Entidad-Relación (Pata de Gallo).
* Identificación de actores, procesos y reglas de negocio.
* Elaboración de documentación funcional y técnica para la presentación del proyecto.

---

##  Módulo E-Commerce

* Catálogo dinámico de productos.
* Gestión de categorías y stock.
* Carrito de compras.
* Procesamiento de pedidos.
* Gestión del estado de las órdenes.
* Experiencia de compra responsive para clientes.

---

##  Módulo de Gestión Logística

* Gestión de choferes y recursos logísticos.
* Asignación y seguimiento de entregas.
* Control del estado logístico de pedidos.
* Monitoreo de operaciones de distribución.
* Administración del flujo de entregas desde la preparación hasta la recepción.

---

##  Gestión de Inventario

* Control de stock de productos.
* Actualización automática de existencias.
* Seguimiento de movimientos de inventario.
* Gestión de disponibilidad para la venta.

---

##  Reportes y Análisis de Datos

* Dashboard administrativo con indicadores de negocio.
* Visualización de métricas de ventas.
* Reportes operativos y estratégicos.
* Análisis de comportamiento comercial.
* Monitoreo del rendimiento de la operación logística.
* Herramientas orientadas a la toma de decisiones basada en datos.

---

##  Auditoría

* Registro de acciones realizadas dentro del sistema.
* Seguimiento de modificaciones sobre datos críticos.
* Control de operaciones administrativas.
* Generación de reportes de auditoría.

---

##  Seguridad y Autenticación

* Gestión de usuarios mediante Clerk Authentication.
* Control de acceso basado en roles.
* Separación de funcionalidades para clientes y administradores.
* Protección de rutas privadas y módulos administrativos.
